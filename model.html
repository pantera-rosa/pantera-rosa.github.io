Implementation

The data we were provided is in a time series format. In order to account for the dimension of time, we utilized a recurrent neural network. RNNs capture time steps in hidden layers, which is critical when trying to analyze how data evolves w.r.t. temporal conditions. In our case, we are dealing with both short and long term time dependencies, where we need to not just capture micro changes per hour but also macro changes like meteorological differences per season. For this, we built an LSTM to capture these long term dependencies specifically.
